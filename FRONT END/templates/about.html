About The Project

Adaptive ML-Based Web Application Firewall with Explainable AI

Traditional Web Application Firewalls (WAFs) depend on static, rule‑based detection mechanisms that struggle to identify evolving cyber threats. This project presents an Adaptive Machine Learning‑Based Web Application Firewall that integrates Explainable AI (XAI) for threat detection.

The system employs multiple machine learning algorithms including XGBoost, LightGBM, and Random Forest to analyze web traffic patterns and detect:

  • SQL Injection (SQLi)
  • Cross‑Site Scripting (XSS)
  • Local File Inclusion (LFI)

Unlike signature‑based approaches, the model learns malicious patterns directly from traffic data, eliminating manual rule maintenance. Using a benchmark intrusion detection dataset from Kaggle, the system extracts network and content‑based features to identify attack signatures.

Dataset

Kaggle Intrusion Detection
15+ numerical features

Algorithms

XGBoost • LightGBM • Random Forest
Ensemble learning

Attacks Detected

SQLi • XSS • LFI

SHAP (SHapley Additive exPlanations) provides transparent, human‑interpretable justifications for each blocking decision, building trust in the AI‑driven security mechanism. A Flask‑based dashboard visualizes attack trends, model confidence scores, and analytical insights for security monitoring.

Built with Python, XGBoost, LightGBM, Random Forest, Scikit‑learn, and SQLite, this solution offers educational institutions and small businesses a cost‑effective, scalable, and interpretable web application firewall that adapts to emerging threat patterns while maintaining transparency through explainable AI.
